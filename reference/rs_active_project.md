# Get 'RStudio' active project

Get 'RStudio' active project

## Usage

``` r
rs_active_project(...)
```

## Arguments

- ...:

  passed to
  [`rs_avail`](https://dipterix.org/dipsaus/reference/rs_avail.md)

## Value

If 'RStudio' is running and current project is not none, return project
name, otherwise return `NA`
