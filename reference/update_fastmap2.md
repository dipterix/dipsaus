# Migrate a `fastmap2` object to a new one

Migrate a `fastmap2` object to a new one

## Usage

``` r
update_fastmap2(from, to, override = TRUE)
```

## Arguments

- from, to:

  `fastmap2` object

- override:

  whether to override keys in `to` if they exist

## Value

Map `to`
